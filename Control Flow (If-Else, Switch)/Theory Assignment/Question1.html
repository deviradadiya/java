What is control flow in JavaScript? Explain how if-else statements work withan example

    Control flow determines the order in which code executes based on conditions.

    If-Else Statement Example:
    
        let num = 10;

        if (num > 0) {
            console.log("Positive");
        } else {
            console.log("Non-positive");
        }

        In this example, if num is greater than 0, "Positive" is logged; otherwise, "Non-positive" is logged.
