What are functions in JavaScript? Explain the syntax for declaring and calling a function.

Functions in JavaScript:
        Functions are reusable blocks of code designed to perform a specific task. They allow modular and maintainable code.
        
        Syntax for Declaring a Function:
        function functionName(parameters) 
        {
            // Code to execute
        }
        
        Syntax for Calling a Function:
        functionName(arguments);
        Example:
        function greet(name) {
            console.log("Hello, " + name);
        }
        greet("Priti"); // Calls the function
