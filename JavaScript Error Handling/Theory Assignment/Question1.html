What is error handling in JavaScript? Explain the try, catch, and finally blocks with an example

Error Handling in JavaScript

Error handling manages runtime errors gracefully using try, catch, and finally blocks.
try: Executes code that may throw an error.
catch: Catches and handles the error.
finally: Executes code regardless of an error.
Example 
 
            try {
                    let result = riskyFunction();
                } catch (error) {
                console.error("Error occurred: ", error);
                } finally {
                console.log("This runs regardless of an error.");
                }
