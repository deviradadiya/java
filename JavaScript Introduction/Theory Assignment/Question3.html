Discuss the use of script tag in HTML. How can you link an external JavaScript file to an HTML document?

The script tag in HTML is used to embed or reference JavaScript code in a webpage. It allows developers to add interactivity, dynamic content, and custom functionality to websites.

  • Placement in HTML Document:
    • Inside : Scripts placed here execute before the HTML is fully loaded. Useful for defining reusable functions. Before the Closing Tag: Recommended for faster page load since it allows the HTML to render before the script runs.
  • Inline JavaScript: You can include JavaScript code directly within the script tag.
  • External JavaScript: The src attribute is used to link an external JavaScript file. This separates JavaScript code from HTML, promoting cleaner and reusable code. inside script tag script src="script.js" close script tag
