Explain the different types of loops in JavaScript (for, while, do-while). Provide a basic example of each.

For Loop: Executes a block of code a specified number of times, using a counter variable.
        for (let i = 0; i < 5; i++) {
            console.log(i);
        }
    
While Loop: Repeats a block of code as long as a specified condition is true.
        let i = 0;
        while (i < 5) {
            console.log(i);
            i++;
        }
    
Do-While Loop: Similar to a while loop, but guarantees execution at least once by checking the condition after the block runs.
        let i = 0;
        do {
            console.log(i);
            i++;
        } while (i < 5);
