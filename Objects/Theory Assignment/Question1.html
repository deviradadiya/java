What is an object in JavaScript? How are objects different from arrays

            
An object in JavaScript is a collection of key-value pairs, where keys are strings (or Symbols) and values can be any data type.


            
Differences between Objects and Arrays:


            
Structure:
 Objects are unordered, using keys, while arrays are ordered, using numeric indices.
            
Use Cases: 
Objects are used for structured data, whereas arrays are used for lists of items.
